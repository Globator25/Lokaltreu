# Project Canvas — Lokaltreu

## Vision
Digitale Stempelkarte als PWA; einfache, sichere Workflows für Kleinstunternehmen. [SPEC]
## Akteure
1) Inhaber/Administrator (Single-Admin-Modell je Mandant), 2) Mitarbeiter (gerätgebunden, kein Login), 3) Endkunde (anonym). [SPEC, ADR 0001]
## Scope
DE/EU Betrieb; Systemsprache Deutsch; Fokus ausschließlich auf Stempel- und Reward-Flows; genau eine aktive Kampagne je Mandant. [ROADMAP]
## Ziele
Onboarding < 5 Min; Mitarbeiter-UI nur zwei Aktionen; Sicherheit durch Automation. [SPEC]
## Erfolgsmessung
SLO 99,90 %, p50≤500 ms / p95≤3 s / p99≤6 s; RPO/RTO 15/60. [ROADMAP]
