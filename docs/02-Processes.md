# Kernprozesse

## Geräteautorisierung
Einmaliger Registrierungslink (TTL ~15 Min), Bestätigung auf Zielgerät, Sicherheits-E-Mail an Admin. [SPEC]
## Stempelvergabe
Mitarbeiter erzeugt QR-Token; Endkunde scannt; ACID-Claim; optional Angebot anzeigen. [SPEC]
## Prämieneinlösung
Endkunde zeigt Redeem-QR; Mitarbeiter bestätigt; Device-Proof erforderlich. [SPEC]
